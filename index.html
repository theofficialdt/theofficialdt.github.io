Frequently Asked Questions

Is this launcher official? ▼

No, Yanix Launcher is an independent, community-driven project and is not affiliated with or endorsed by YandereDev.

Is this a native port of the game? ▼

Not exactly. It uses WINE to run the Windows version of Yandere Simulator on Linux and macOS without modifying the game's code.
